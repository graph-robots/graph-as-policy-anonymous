# GaP — Graph-as-Policy · anonymous project page

Project page for **GaP: A Graph-as-Policy Multi-Agent Self-Learning Harness
for Variational Automation (VA) Tasks**, served at
<https://graph-robots.github.io/graph-as-policy-anonymous/>.

> Authors, affiliations, and analytics are anonymized for double-blind review.
> Code, skill library, and docs links are disabled until the review period ends.

Static site — no build step. `index.html` + `assets/` is everything. The
`.nojekyll` file keeps GitHub Pages from mangling asset paths.
